# Inputs

## For AI to Generate PHP Code

- Feature description or user story
- Existing codebase patterns (if extending)
- Database schema (if DB changes needed)
- UI requirements (Bootstrap, layout)
- Auth requirements (public, logged-in, role-based)

## For AI to Debug or Review

- PHP error messages
- XAMPP error log entries
- Database error details
- Web server response (404, 500, etc.)
- Browser console errors
- File paths and folder structure

## Common XAMPP Inputs

```
XAMPP Control Panel status
  - Apache: Running / Stopped
  - MySQL:  Running / Stopped
  - PHP version: 8.x
  - MySQL port: 3306 / custom

phpMyAdmin: http://localhost/phpmyadmin
Project URL: http://localhost/project-name
```
