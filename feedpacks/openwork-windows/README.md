# OpenWork на Windows

**Полное пошаговое руководство по установке и настройке OpenWork AI Agent на Windows 10/11.**

> От свежей Windows до полностью рабочего OpenWork агента.
> Гайд не имеет аналогов — собран из реального ежедневного опыта эксплуатации.

## Назначение

Помочь разработчикам и пользователям быстро настроить OpenWork на Windows
со всем необходимым окружением: Python, Tesseract OCR, Git, AI-провайдер,
разрешения и типовые проекты.

## Кому подойдёт

- Разработчики, впервые устанавливающие OpenWork
- Пользователи, переходящие с macOS/Linux на Windows
- Команды, настраивающие окружение для AI-агентов
- Все, кто хочет избежать типичных ошибок Windows-настройки

1. Откройте [WINDOWS_SETUP.md](WINDOWS_SETUP.md) для полного руководства
2. Следуйте шагам по порядку — от Python до проверки стека
