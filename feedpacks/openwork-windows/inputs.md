# Inputs — что нужно знать перед началом

## Обязательно

| Данные | Пример | Где взять |
|--------|--------|-----------|
| Имя пользователя Windows | `ppronin` | `$env:USERNAME` |
| Версия Windows | Windows 11 Pro 23H2 | `winver` |
| Версия PowerShell | 5.1 | `$PSVersionTable.PSVersion` |
| Диск для проектов | `E:\` | Свободное место > 10 ГБ |
| Путь к рабочей директории | `E:\ai\openwork` | Свой выбор |
| Нужен ли Tesseract | Да / Нет | Если планируется OCR |
| AI-провайдер | Anthropic / OpenAI / OpenRouter / Ollama | Ваш выбор |

## Опционально

| Данные | Зачем |
|--------|-------|
| Нужен ли Node.js | Для MCP-серверов на JavaScript |
| Путь к XAMPP | `E:\xampp\htdocs` — если есть веб-проекты |
| Репозиторий GitHub | Для клонирования проектов |
| API-ключ Anthropic | `sk-ant-...` для Claude |
| API-ключ OpenAI | `sk-...` для GPT |

## Ожидаемый уровень пользователя

- Базовое знакомство с командной строкой
- Умение копировать и вставлять команды
- Права администратора Windows (для winget, установки)
