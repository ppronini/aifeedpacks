# PHP Best Practices FeedPack

**Category:** Development  
**Version:** 1.0.0  
**Description:** Ensures AI follows modern PHP standards, security best practices, and clean code principles when generating or reviewing PHP code.

## Purpose

Provide AI with a consistent set of PHP development rules so output is secure, readable, and maintainable.

## Usage

1. Copy this folder into your AI project context
2. Or feed the files directly into your AI tool (ChatGPT, Claude, Cursor, etc.)
3. Reference `instructions.md` as the primary system prompt

## Intended Audience

- PHP developers
- Teams doing code review
- AI-assisted PHP development
