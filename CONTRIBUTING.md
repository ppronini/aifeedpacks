# Contributing to AI FeedPacks

Thank you for contributing! 🎉

## How to Submit a FeedPack

1. Fork this repository
2. Create a new folder under `feedpacks/your-feedpack-name/`
3. Include the following files (use [PHP Best Practices](feedpacks/php-best-practices/) as a template):

```
your-feedpack-name/
├── README.md          — Overview, purpose, audience
├── instructions.md    — AI role, behavior, rules
├── workflow.md        — Step-by-step process
├── inputs.md          — Required information
├── output-format.md   — Expected output structure
├── examples.md        — Good and bad examples
├── constraints.md     — Hard rules
├── checklist.md       — Quality validation
└── changelog.md       — Version history
```

4. Update the table in `README.md` with your FeedPack
5. Open a Pull Request

## Guidelines

- Use clear, concise English
- Keep files focused on one topic
- Follow the standard file structure
- Add practical examples
- No marketing fluff — just useful content
